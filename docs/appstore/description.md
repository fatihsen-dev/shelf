## Description

```
Shelf is a fast, native clipboard manager for macOS. Built with pure AppKit for minimal CPU and memory footprint.

FEATURES
• Universal clipboard history — text, links, colors, images, and files
• Global hotkey opens the picker at your cursor
• Paste selected items into the active application with one keystroke
• Incremental search across your entire history
• Pin favorite items so they never expire
• Menu bar access for quick previews
• Persistent history that survives restarts
• Native performance — no Electron, no background polling

PRIVACY
Shelf stores all clipboard data exclusively on your Mac. Nothing is sent to remote servers. There is no analytics, no telemetry, and no account system.

REQUIREMENTS
Shelf requires Accessibility permission to register the global hotkey and paste into the frontmost application.
```
